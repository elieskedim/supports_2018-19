La grille CSS ou "CSS Grid"

La Grille Fractions Zoning

Le système de grille CSS s'appuie sur les LIGNES verticales et horizontales qui dessinent ma grille, pas sur les rangs ni les colonnes !

Le système de grille CSS permet de positionner des éléments HTML

  • en colonnes : grid-template-columns: 100px 100px 100px;
  • et en rangs : grid-template-rows: 100px 100px 100px;
